# tampermonkey-jstools
油猴/篡改猴小工具集合

## 简介
这是一个油猴脚本(Tampermonkey UserScript)集合项目，提供各种实用的浏览器扩展功能。

## 使用方法
1. 首先需要安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击需要安装的脚本链接，即可自动安装
